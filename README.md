#**Final STA130 Project**:
A Comparative Analysis of Electricity Accessibility Across African Countries
Originally done on JupyterHub

## Overview 
In this project, we analyzed electricity systems across African countries using real-world data. We explored how factors such as infrastructure expansion, economic development, and poverty are associated with better electrification outcomes.
Our goal was to better understand what are the main factors that contribute to accessible electricity systems in Africa.

## Statistical techniques in R used
- Data cleaning and transformation (dplyr)
- Data visualization (ggplot2)
- Linear regression models
- Grouped summary statistics
- Hypothesis testing

## Future Improvements
- Use multivariable models to account for more factors
- Analyze trends over time instead of static-ly
- Add additional variables (e.g., geography, policy, energy sources)
